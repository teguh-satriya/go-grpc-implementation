# Simple GO gRPC Implementation
This repository created to research implementation of gRPC in Go. This also will be research repository for researching other stuff like unit test, genereic implementation etc. 

## Requirements
* [Go](https://go.dev/)
* [Buf CLI](https://docs.buf.build/installation) to generate protobuf
* [Mockery](https://github.com/vektra/mockery) (maybe for researching unit test)

## How To Run
Run `go run main.go` on terminal
